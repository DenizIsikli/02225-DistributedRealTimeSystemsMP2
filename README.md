# TSN Credit-Based Shaper Analysis: CBS vs Strict Priority

This project implements worst-case delay (WCD) analysis, discrete-event simulation, and comparison of Credit-Based Shaper (CBS) and Strict Priority (SP) scheduling for periodic real-time streams in a Time-Sensitive Networking (TSN) context.

---

## Setup and Installation

### 1. Prepare Test Case Data

Place a test case folder in the root directory of the project:

mini-project-2-RTDS/
├── test-case-1/
│   ├── test-case-1-routes.json
│   ├── test-case-1-streams.json
│   ├── test-case-1-topology.json
│   └── test-case-1-WCRTs.csv
├── src/
├── context/
└── README.md

### 2. Install Dependencies

This project uses only the Python standard library.
Requires Python 3.8+.

No external packages required.
